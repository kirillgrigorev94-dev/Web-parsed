1. Какие методы BeautifulSoup используются для поиска элементов?

Используемые методы:
find() - находит первый элемент соответствующий категориям
find_all() - возвращает список всех элементов подходящих под условие
select() - ищет элементы по CSS-селекторам
get_text() - извлекает текстовое содержимое элемента
get() - получает значение атрибута элемента

2. Почему важно использовать User‑Agent при парсинге?

User‑Agent помогает обойти блокировки некоторых сайтов или подозрительных клиентов
Имитирует запросы от реального браузера, а не от скрипта
Снижает риск распознования и блокировки парсера как автоматизированной системы

3. Как обрабатываются исключения при парсинге?

Используются:
блоки try-except (например AttributeError если элемент не найден)
проверка кода ответа HTTP (status_code) перед обработкой данных
повторные попытки запроса при сетевых ошибках
логирование ошибок для последующего анализа

4. Какие существуют способы оптимизации при работе с большими объёмами данных?

Ключевые методы:
добавление пауз между запросами (time_sleep) для снижения нагрузки на сервер
порционная обработка данныхиспользование эффективных парсеров
кеширование ответов для повторного анализа без новых запросов
применение асинхронных запросов для ускорения сбора данных
ограничение глубины парсинга (например только первые N страницы)
